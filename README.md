# ASP.NET Core Web Application

Αυτό το αποθετήριο περιέχει μια διαδικτυακή εφαρμογή (Web Application) αναπτυγμένη σε περιβάλλον **.NET**[cite: 12].

## Δομή Project
Το project ακολουθεί τη δομή του Visual Studio και περιλαμβάνει:
* **Solution File**: Για τη διαχείριση των εξαρτήσεων του project[cite: 12].
* **WebApplication1**: Ο κύριος φάκελος της εφαρμογής με τον κώδικα (C#, HTML, Razor/Controllers)[cite: 12].

## Τεχνολογίες
* **C# / .NET**: Backend λογική.
* **ASP.NET Core**: Framework για την ανάπτυξη web εφαρμογών.
* **Visual Studio**: Περιβάλλον ανάπτυξης (IDE).

## Πώς να το τρέξετε
1. Βεβαιωθείτε ότι έχετε εγκατεστημένο το **.NET SDK**.
2. Ανοίξτε το αρχείο λύσης (`.sln`) με το Visual Studio[cite: 12].
3. Πατήστε `F5` ή το κουμπί **Run** για να ξεκινήσει ο τοπικός διακομιστής.
